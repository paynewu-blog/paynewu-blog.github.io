常用快捷键

Window

Alt + → ：代码前进下一级

Alt + ← ：代码返回上一级

Ctrl + K + 0: 收缩所有代码(0是代码折叠级别，同理可以换用1，2，3等)

Ctrl + K + J: 取消收缩所有代码

Ctrl + K + S：快捷键文档

Alt + Shift + F ：代码格式化

Ctrl + Shift + P： 显示所有命令

Ctrl + P： 显示所有命令

Mac

  • Ctrl + -：代码返回上一级

  • Ctrl + Shift + -： 代码前进下一级

  • Command + K + 0: 收缩所有代码(0是代码折叠级别，同理可以换用1，2，3等)

  • Command + K + J 取消收缩所有代码

  • Command + K + S：快捷键文档

  • Opt + Shift + F ：代码格式化

  • Command + Shift + P： 显示所有命令

  • Command + p ：打开最近文件
